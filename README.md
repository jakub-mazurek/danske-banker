# Danske Banker
This repo contains models explaining cooperation between Danish banks and the unique decision to found IT companies together with their competitors.

## Data

- KPI_Penge_fonds_Real_202412.xlsx from Finanstilsynet (Danish FSA): https://www.finanstilsynet.dk/tal-og-fakta/statistik/noegletalsdatabasen
- BankByDC.xlsx: own work based on datacentrals websites and yearly reports
